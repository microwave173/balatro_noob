# Balatro White Stake Reference (Stable Baseline)

This file is a permanent compact reference derived from `balatro_white_stake_guide_en.md`.
It is intended to stay stable and provide long-term policy grounding.

## Objective
- Clear White Stake Ante 8 boss reliably, not endless high-score greed.
- Favor consistency and survival over flashy but unstable lines.

## Scoring Model
- Core formula: score is driven by chips and mult.
- Late game consistency usually needs all three lanes: chips source + mult source + xmult path.
- If only one lane grows, scaling usually stalls by mid/late antes.

## Early Game (Ante 1-2)
- Prioritize immediate survival and tempo.
- Default to easy, stable hands when uncertain: flush/pair/two-pair lines.
- Do not over-force straights if outs are poor.
- Aim to secure at least one direct scoring Joker by early-mid game.
- Avoid aggressive resets unless the run has no scoring direction after multiple shops.

## Shop Priorities
- Buy order baseline:
  - Immediate scoring Joker
  - Strong economy support
  - Planets/Tarots/Spectral that reinforce your current line
  - Utility pieces
- Avoid spending all money every shop.
- Skip low-impact buys to preserve future spike potential.
- Avoid reroll spam before core scoring is established.

## Economy Rules
- Keep a money buffer when possible.
- Spend aggressively only on clear power spikes.
- If shop value is weak, advancing is often better than forcing buys.

## Hand Selection & Discards
- Play beatable lines first; avoid unnecessary high-variance plays.
- If blind is clearly beatable now, do not burn discard/hands for marginal optimization.
- Use discard when current hand is weak and there is realistic improvement.
- Preserve at least one fallback line for bad draws.

## Mid Game (Ante 3-5)
- Lock one primary scoring direction; avoid fragmented pivots.
- Build around broad-value Jokers over narrow combo traps.
- Upgrade hands that you can produce consistently, not just occasionally.
- Start planning xmult integration before late-game wall.

## Late Game (Ante 6+)
- Prioritize scaling consistency and boss-safe play.
- Protect strong engine pieces; avoid risky detours without clear payoff.
- Keep decisions simple under uncertainty: stable score > speculative greed.

## Safety / Agent Stability
- Prefer valid, conservative actions over edge-case sequences.
- If uncertain between similar EV choices, choose the safer and more deterministic action.
